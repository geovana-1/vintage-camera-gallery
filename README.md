# Vintage Camera Gallery
Uma aplicação web front-end desenvolvida para exibir uma galeria interativa de máquinas fotográficos clássicas. 

# Sobre o Projeto
Este projeto foi construído com foco em boas práticas de desenvolvimento front-end, aplicando conceitos de organização de arquivos, componentização e manipulação do DOM. A escolha do tema reflete uma paixão particular pela estética analógica e pelo charme das câmeras vintage.

# Tecnologias Utilizadas
- **HTML5:** Estruturação semântica de múltiplas páginas.
- **CSS3:** Estilização e design responsivo.
- **Vanilla JavaScript:** Lógica de interface e manipulação de elementos.

# Estrutura da Aplicação
- `public/`: Contém os arquivos HTML de cada página.
- `src/components/`: Scripts com a lógica da galeria.
- `src/assets/`: Imagens e folhas de estilo.

# Como executar
1. Clone este repositório.
2. Abra o arquivo `public/index.html` em seu navegador.
